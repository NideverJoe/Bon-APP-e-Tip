# Bon-APP-e-Tip

https://nideverjoe.github.io/Bon-APP-e-Tip/

## Purpose
#### Mobile Ready Tip Calculator

## Technologies Used
* JQuery
* Bootstrap

## Overview
* User inputs:
** Currency Type
** Total Bill
** Number of Guests
** Tip Desired
* App Displays Result
** Total Bill
** Total Tip
** Bill Split
** Tip Split
